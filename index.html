Estrutura HTML

Nesse primeiro modulo de aprendizado do curso eu pude ter conhecimento em como funciona a estrutura do html desde o inicio que consiste na abertura e fechamento da tag principal < html >, desde onde se inicia em < head >, onde podemos definir o titulo do site que aparece nas guias do navegador dentre outras funções que definem o site.. Posteriormente temo a função < body >, onde está localizado todo o corpo do website e conteúdo atrativo para o visitante.

Tags

As tags são responsáveis ​​por criar elementos nas páginas web e ter a função de organizar, identificar e limitar o conteúdo de uma página. Cabe então ao browser considerar esses elementos e renderizá-los apresentando seu conteúdo aos usuários.

    Podemos ver o exemplo da tag < h1 > onde é utilizada para determinar o principal assunto abordado em uma página, no caso o Titulo de tópicos por exemplo.

    Temos a < ol > que determinam listas ordenadas especificamente e mais utilizado para tópico de 1,2,3 até quantidade que desejar de tópicos

    Conhecemos a < ul > que também cria listas, porém desordenadas para quem deseja ter listas de as ordens numericas.

    A tag < p > como a própia ja indica na inicial, ela cria paragráfos para criação de textos e mensagens

Atributos das tags

    Os atributos são informaçõs passadas dentro da tag para que ela haja como desejamos.

    Normalmente é verificado através do nome ou valor.

    Um exemplo são as "classes" e o "id" que informamos atraves de nome ou valor, por exemplo: < h2 id="atributo" > < /h2 >.
